TagRigger

TagRigger is a productivity tool for organizing and managing a catalog of text-based tags. The application allows users to create tag sets. To test the app, please add several tag sets and add individual tags to those sets. You can edit, rename, and delete both the sets and the tags.

The application is a standalone data manager and requires no external files, accounts, or specialized hardware for testing.
